# 数据集说明

由于数据集文件较大，仓库中不直接存放完整数据文件。需要复现本项目的用户，可在本仓库的 Release 页面下载数据压缩包，并按照 README 中的运行说明导入 MySQL 后执行 SQL 脚本。

## 数据字段

本项目使用的用户行为数据主要包含以下字段：

| 字段名 | 含义 |
|---|---|
| `user_id` | 用户 ID |
| `item_id` | 商品 ID |
| `category_id` | 商品类目 ID |
| `behavior_type` | 用户行为类型 |
| `time_stamp` | 用户行为发生时间戳 |

## 行为类型说明

| 行为类型 | 含义 |
|---|---|
| `pv` | 浏览 |
| `fav` | 收藏 |
| `cart` | 加购 |
| `buy` | 购买 |

## 使用说明

1. 前往本仓库的 Release 页面下载数据集压缩包。
2. 解压数据文件。
3. 将数据导入 MySQL，生成原始表 `userbehavior`。
4. 按照 README 中的运行说明执行 SQL 脚本。
